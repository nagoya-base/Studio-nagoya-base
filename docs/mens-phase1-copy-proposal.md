## 8. 特徴セクション全文（確定版・6項目）

現行の「講習会・ワークショップを開ける」を削除し、以下の6項目に置き換える。

1. **男性同士で使える**
   男性ペアや男性グループでの緊縛練習・撮影を歓迎しています。ゲイ・バイ男性を含め、性別や性的指向を問わずご利用いただけます。

2. **人目を気にせず使える**
   完全個室・無人入室。他の利用者やスタッフと対面することなく、衣装やロープ表現に集中できます。

3. **本格的な吊り設備**
   木組みの吊り床と竹吊りを完備。床縄の練習から、経験者による空中緊縛撮影まで対応しています。

4. **フェティッシュ撮影に使える**
   ユニフォーム、制服、スポーツウェア、レザーなど、衣装や身体表現を生かした作品撮りに利用できます。

5. **スマホでも撮れる**
   フルカラー天井照明、白・黒背景紙、パネルライトを常設。スマートフォンでも照明を生かした撮影ができます。

6. **必要な道具が揃う**
   麻縄、スリング、カラビナ、ヨガマット、置き畳など、緊縛練習や撮影に必要な基本備品を常設しています。

修正理由：「講習会・ワークショップを開ける」はスタジオ運営者が指導するように誤認されるため削除。「吊れる」の訴求は3番に残す。4番は特定衣装の断定的な列挙を避けた。

## 13. 画像alt文の方針（Phase 2以降の実装ルール）

Phase 1では画像差し替え・alt属性変更は行わないが、今後の男性作例用alt文の方針を以下の通り定める。

**避けるべき例**（現行ページに存在）：
```
Studio Nagoya Base gallery image 1
Studio Nagoya Base gallery image 2
```

**望ましい例**：
```
男性モデルを使った緊縛作品撮りの作例
吊り床を使用した男性同士の緊縛撮影
フルカラー照明を使った男性フェティッシュ撮影
スポーツユニフォームを着た男性モデルの作品撮り
```

原則：画像から確認できない性的指向や関係性をalt文で断定しない。
