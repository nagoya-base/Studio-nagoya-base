## 1. 市場構造の分析

### 1-1. 緊縛需要は「PR不足」ではなく「構造的天井」

緊縛撮影の需要が頭打ちなのはPR次第ではなく、以下の構造的理由による。

| 制約要因 | 内容 |
|---|---|
| 自己開示コスト | BDSM愛好者であることを公開しにくい。口コミが広がらない |
| 縄師ボトルネック | スタジオ以前に縄師確保が最大障壁。スタジオ予約を増やしても縄師供給が追いつかない |
| SNS制限 | 緊縛コンテンツはXでもセンシティブ扱い。アルゴリズム露出が構造的に不利 |
| 利用頻度の低さ | 月1回未満が大半。コスプレの月複数回と対照的 |
| 競合の不在 ≠ 需要の存在 | 名古屋に緊縛専用スタジオが公開されていないのは需要が地下に潜っているためであり、市場規模が大きいわけではない |

**結論：PRを強化しても緊縛だけでの月15件達成は名古屋規模では非現実的。上限は月5〜7件と推定。**

### 1-2. 上前津という立地の真の価値

- 上前津駅 = 大須商店街の玄関口（南東側出口、徒歩5分で万松寺通）
- 大須の人流：**平日3万人/日、土日7万人/日**
- **世界コスプレサミット**（毎年8月、名古屋開催）：3日間25万人超。大須がメイン会場・パレードルート
- 国内登録コスプレイヤー：約34万人。名古屋は世界最大級のコスプレ集積都市

この立地を緊縛専用で使っているのは著しいアンダーユーティライゼーション。

### 1-3. コスプレ市場の特性

- コスプレイヤーの**70.2%が「撮影場所がない」を不満**として挙げている（2019年調査）
- 月収の21〜30%を撮影・コスプレに支出する層が最多（29.7%）
- 緊縛の一回性に対し、コスプレは**リピーター型**（月複数回利用）

### 1-4. 価格競争力の整理

| | 当スタジオ（現行） | 大須観音駅圏の市場平均 |
|---|---|---|
| 時間単価 | **¥3,250/時間**（土日3h） | **¥1,313/時間** |

一般コスプレ向けには2.5倍高い。**価格プレミアムを正当化できる差別化**が必要。

## 2. 顧客セグメント戦略

### 推奨セグメント構成（月15件目標）

| セグメント | 月件数 | 平均単価 | 月収 | 価格正当化の根拠 |
|---|---|---|---|---|
| 緊縛コア | 4〜5件 | ¥11,500（3〜4h混合） | ¥52,000 | 専用設備・プライバシー・希少性 |
| **R18コスプレ・ファンコンテンツ** | 5〜6件 | ¥9,750（3h土日） | ¥52,000 | 他に行く場所がない（競合ほぼ皆無） |
| 一般コスプレ・ポートレート | 4〜5件 | ¥8,000（平日・会員混合） | ¥36,000 | 大須至近・完全個室 |
| **合計** | **13〜16件** | — | **¥140,000** | |

### なぜR18コスプレがキーセグメントか

```
一般コスプレスタジオ  →  R18衣装・露出系・バインデージコスプレ  →  ほぼ全滅
緊縛専用スタジオ     →  緊縛以外は想定外
当スタジオ           →  両方に対応できる（大須徒歩圏）＝競合がほぼ存在しないニッチ
```

- 価格プレミアム（¥9,750/3h）が正当化される
- 緊縛ユーザーと同じプライバシーニーズを持つ
- Xでの直接集客が主チャネル（一般広告では獲得困難）

## 3. チャネル戦略

### 3-1. チャネル別の役割分担

| チャネル | 表現方針 | 獲得ターゲット |
|---|---|---|
| **自サイト（LP）** | 入口は一般向け・完全個室撮影スタジオとして訴求。緊縛・R18は奥で訴求 | 全セグメント |
| **SpaceMarket** | 「完全個室・撮影機材完備の撮影スタジオ」として掲載。成人向け可とは書かない。吊り床設備の記載は避ける | 一般コスプレ・ポートレート |
| **X（Twitter）** | センシティブ設定でR18コスプレ作例を投稿。緊縛作例も継続 | R18コスプレ・緊縛コア |
| **大須イベント連動** | 世界コスプレサミット期間中のローカル告知・配布 | 一般〜R18コスプレ |

### 3-2. SpaceMarket掲載の要点

- タイトル例：「【完全個室】大須徒歩5分の撮影スタジオ｜機材無料｜コスプレ・ポートレート対応」
- 成人向け可能の旨はSpaceMarket上には記載しない（規約上グレー）
- 実際の成人向け利用は自サイト経由の直接予約で完結させる

## 4. LP改修方針（Issue #16 実装計画）

### 基本原則：「入口は広く、奥で緊縛・R18を訴求」

### Group A：SEOメタデータ（最優先・新規流入創出）

| 項目 | Before | After |
|---|---|---|
| title | 名古屋・上前津の緊縛撮影スタジオ｜Studio Nagoya Base | 名古屋・上前津の完全個室撮影スタジオ｜コスプレ・ポートレート・作品撮り｜Studio Nagoya Base |
| meta description | 緊縛撮影に特化した〜 | 名古屋・上前津の完全個室撮影スタジオ。コスプレ・ポートレート・白背景撮影に対応。スマートフォンでも本格撮影可。撮影機材無料貸し出し。緊縛撮影にも対応。 |
| og:title / twitter:title | 緊縛撮影スタジオ〜 | 完全個室撮影スタジオ〜 |
| og:description / twitter:description | 緊縛撮影に特化した〜 | コスプレ・ポートレート・白背景撮影に対応した完全個室スタジオ〜 |

### Group B：ヒーロー（ファーストビュー）

| 項目 | Before | After |
|---|---|---|
| h1 | 完全個室、誰にも見られない。 | 完全個室の撮影スタジオ。 |
| サブコピー | 2人の時間を邪魔しない場所。 | コスプレ・ポートレート・作品撮りに。名古屋・上前津。 |

### Group C：用途セクション（Issue #16の核心）

Before（緊縛ファースト）:
- 緊縛の練習 / 撮影（静止画・動画） / アート・身体表現 / ワークショップ / ペア・グループの技術練習

After（コスプレ・ポートレートファースト）:
- コスプレ撮影 / ポートレート撮影 / 白背景・物撮り / 作品撮り・ファンコンテンツ / 撮影（静止画・動画）/ アート・身体表現 / **緊縛撮影・練習にも対応** / ワークショップ

セクションタイトル：「こんな用途に」→「こんな撮影に使えます」

|---|---|---|

### Group E：ギャラリー注釈

- 「実際の作例は Twitter で〜」→「コスプレ・ポートレートなど様々な作例は X で〜」

- `<strong>` タグを外して緊縛注記の視覚的優位を下げる

- 「撮影や緊縛・創作表現を安全に〜」→「撮影・創作表現を安全に〜。緊縛撮影にも対応しています。」
